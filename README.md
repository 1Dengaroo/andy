# andy

Personal portfolio site. Visit at [andydeng.me](https://andydeng.me).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- **Next.js 15** (App Router) + **React 19** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui** for styling and components
- **Three.js** for animated backgrounds
- **Zod** + **react-hook-form** for the contact form
- Deployed on **Vercel**

## Theming

6 built-in themes (Light, Dark, Minecraft, Dotcom, Comic, Claude) powered by a CSS custom property token system. See [docs/THEME_FRAMEWORK.md](docs/THEME_FRAMEWORK.md) for details.
